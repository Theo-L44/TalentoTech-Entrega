# 🎲Rincón Lúdico
Sitio web desarrollado con HTML, CSS y Javascript para un comercio de juegos de mesa

## 📌Características
- Estructura en HTML5
- Catálogo de juegos de mesa y accesorios
- Página de contacto
- Carrito de compras

## 🛠️Tecnologías
* HTML5
* CSS3
* JAVASCRIPT
